## Overview

Boris Sprickut provides a centralized space for your project documentation. You create, organize, and collaborate on docs without the hassle of scattered files or complex tools. Start building your knowledge base today.

<Callout kind="info">
  Boris Sprickut supports Markdown, MDX components, and version history for seamless documentation workflows.
</Callout>

## What is Boris Sprickut?

Boris Sprickut acts as your default documentation hub. You manage all project-related guides, APIs, changelogs, and FAQs in one place. It integrates MDX for rich, interactive pages with components like tabs, steps, and code examples.

Key aspects include:
- Intuitive editor for quick doc creation
- Automatic navigation and search
- Customizable themes and branding

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://borissprickut.com` and create your free account.
  </Step>
  <Step title="Create Workspace" icon="folder-plus">
    Set up a new documentation space for your project.
  </Step>
  <Step title="Add First Page" icon="file-plus">
    Click "New Page" and start writing with Markdown or MDX.
  </Step>
  <Step title="Publish" icon="globe">
    Save and share your live documentation link.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Embed Boris Sprickut docs in your app
  const embedDocs = (url) => {
    const iframe = document.createElement('iframe');
    iframe.src = url;
    document.body.appendChild(iframe);
  };
  embedDocs('https://docs.example.com/introduction');
  ```
  ```python
  # Fetch latest docs via API
  import requests
  response = requests.get('https://api.example.com/docs/introduction')
  print(response.json())
  ```
